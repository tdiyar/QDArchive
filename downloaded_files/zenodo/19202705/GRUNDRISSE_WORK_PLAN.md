### Operation G1: THE SELF-LEGITIMATION CUT (~5–8K words removed)

**The problem:** The book over-annotates its own importance. "Deepest claim," "most important," "strongest objection," "this is not metaphor," "this is the real engine," "foundational," "decisive." Repeated self-marking reduces force. The rule: for every three places the book says "this is the deepest/strongest/decisive move," cut two.

**Method:** Run a grep pass for the following markers across the full monograph:

```
deepest, strongest, most important, decisive, foundational, 
this is not metaphor, this is the real, not merely, 
unprecedented, most significant, the true engine,
the real claim, at its most, the book's greatest
```

For each hit: is this the ONE time the phrase earns its place in this notebook? If yes, keep. If it's the second or third occurrence of the same self-marking gesture, cut or cool it. Replace with placement confidence — let the section's position in the architecture carry the weight.

**Blog mining target:** None. This is editorial surgery.

**Estimated net change:** -5,000 to -8,000 words. Some of these phrases will be replaced by single transitional sentences, so net loss is ~4,000–6,000.
