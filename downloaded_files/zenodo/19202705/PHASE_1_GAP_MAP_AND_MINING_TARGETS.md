# PHASE 1: GAP MAP + ANTI-REGULARITY AUDIT + ARCHIVE MINING TARGETS

## Governing Principle

The monograph is 164,860 words of genuine force. The regularity problem
is no longer "it lacks content." The regularity problem is: the content
arrives in the same mode too continuously. The gap map below identifies
not just WHAT is missing but WHAT FORM the missing material must take
to break the specific regularity of each notebook.

# ANTI-REGULARITY AUDIT: THE 10 WORST REMAINING REGULARITIES

1. **NB I §4–§10: Seven sections, same cadence.** Define → explain →
   example → exercise, repeated. BREAK by varying section lengths
   radically and inserting a scene between §6 and §7.

2. **NB II operators at equal weight.** σ_S gets a full derivation.
   All others get mini-derivations. σ_M should be NEARLY as large
   as σ_S. σ_T and σ_sw should be compressed. The weight variation
   IS the hierarchy.

3. **NB III's five graphs, same format.** Context → graph → measurement
   → finding, five times. Graph 3 should be PURE DIAGRAM. Graph 5
   should be PURE LEDGER. At least one should be interpersonal.

4. **NB IV §4–§5e post-Cathedral letdown.** The Cathedral is the peak.
   Everything after must not return to business-as-usual. Insert
   a prose-performance of the value-form. Insert more Marx German.
   Make the post-Cathedral sections feel EARNED, not appended.

5. **NB V cases at uniform genre.** Eleven cases, same analytical format.
   O'Keeffe should be curatorial. Twenty-Dollar Loop should be field
   report. Google Projectionist should be technical autopsy. The medium
   erasure should be documentary. The saint icon should be liturgical.

6. **NB VI Protocol→Failure→Example×3.** Same sequence for all three
   protocols. Protocol II should be TERSE. Protocol III should be
   INSTITUTIONAL. Protocol I already has the gap disclosure.

7. **NB VII §1–§4 orderly propagation theory.** The notebook about
   hostile infrastructure should feel hostile. Reorder: start with
   the kill box (§4b), THEN build the theory from inside the danger.

8. **NB VIII recovers too sequentially.** σ_κ fails → Cleis → Jot &
   Tittle → Elijah → Evil → Somatic → Exhaustion. Too orderly.
   Some of this material should arrive BEFORE the failure. Some
   should be INTERRUPTED by the failure.

9. **NB IX fragments too well-curated.** Even the "broken" fragments
   are artful. Add truly rough material from the archive. Add
   material that predates the framework.

10. **ACROSS ALL: exercises arrive too predictably.** Currently: end
    of section → exercise. Some exercises should arrive MID-SECTION.
    Some should arrive with no framing. Some should feel like traps.
