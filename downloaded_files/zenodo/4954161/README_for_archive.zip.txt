This repository contains data and code associated with the article: 

Bragg JG, Potter S, Afonso Silva AC, Hoskin CJ, Bai BYH, Moritz C
Phylogenomics of a rapid radiation: the Australian rainbow skinks


The raw sequence data associated with this article are available on the NCBI Sequence Read Archive (SRA; http://www.ncbi.nlm.nih.gov/sra/). All are in SRA project PRJNA289283. Table S1 of the article provides BioSample numbers.  

This archive contains two directories, containing code and data:

1. code -----------------------------

clean         -- perl and shell scripts to clean raw sequence data 
assemble      -- perl and shell scripts used to assemble sequences
                from clean sequence reads   

-------------------------------------


2. data ------------------------- 

Locus information and assembled sequence data:
 
lists         -- list of complete loci, and complete and informative loci
phy           -- sequence alignments used in analyses
etc           -- an example of an xml file used in StarBEAST2 analyses

-------------------------------------


Scripts are provided here primarily for archival purposes. See also:
https://github.com/MVZSEQ
https://github.com/jasongbragg/exon-capture-phylo
